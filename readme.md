## webpack3-feature

A repository for you to understand diff feature for webpack3 versus webpack2

## How to run 

```bash
    # install dependencies
    npm run i
```

## run webpack2 repo

```bash
    cd webpack2
    npm start
```

## run webpack3 repo

```bash
    cd webpack3
    npm start
```
